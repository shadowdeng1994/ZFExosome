This directory contains the relevant scripts for processing next-generation sequencing (NGS) data.

The following files are included:

* **genome.callSNPs.pipeline.sh** : variant calling workflow from genome next-generation sequencing data
* **mRNA.callSNPs.pipeline** :  variant calling workflow from bulkRNA and smartseq2 next-generation sequencing data
* **callSNPs.sh** :  GATK4 HaplotypeCaller command parameter
